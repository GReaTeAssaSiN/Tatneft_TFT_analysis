# Прогнозирование продаж АЗС Татнефть — TFT модель + EDA / Прогнозы / Интерпретация

Финальная работа по курсу **Технологии и Архитектуры Больших Данных**.  
Разработка модели на базе Temporal Fusion Transformer (TFT) для прогнозирования продаж топлива и сопутствующих товаров сети АЗС Татнефть.

---

## Задача

Построить мультицелевую модель временных рядов, которая по 7 суткам ретроспективных данных предсказывает на 24 часа вперёд:
- **7 видов топлива**: АИ-92, АИ-95, АИ-98, ДТ Евро+, ДТ ТАНЕКО, ДТ Летнее, ДТ Зимнее
- **5 категорий магазина**: напитки, закуски, автотовары, кофе, табак

Модель учитывает рекламу, акции, трафик по типу транспорта, погоду, цены конкурентов и статические характеристики каждой АЗС.

---

## Данные

5 АЗС Татнефть, почасовые данные за 2023 год (43 800 строк):
- `5stations_metadata.csv` — паспорт АЗС: тип дороги, услуги, количество колонок
- `5stations_data.csv` — погода, трафик, продажи, цены, акции, реклама

## Архитектура TFT

Temporal Fusion Transformer (Lim et al., 2020) принимает три типа входов:

| Тип | Примеры | Кол-во |
|---|---|---|
| Статические (metadata АЗС) | тип дороги, кол-во колонок, площадь магазина | 30 (3 кат. + 27 числ.) |
| Известные будущие | час суток, день недели, праздники, цены топлива, акции | 32 (4 кат. + 28 числ.) |
| Наблюдаемые прошлые | погода, трафик, цены конкурентов, выручка магазина | 31 (включая 12 целевых) |

Горизонт прогноза: **24 часа**. Ретроспектива: **168 часов (7 суток)**.

## Препроцессинг

1. Заполнение пропусков: `holiday_name → "нет_праздника"`, `ad_channel → "нет_рекламы"`
2. Winsorization выбросов по IQR (строки не удаляются — временной ряд непрерывен)
3. Label Encoding категориальных переменных
4. Циклическое sin/cos кодирование: hour, day_of_week, month, week_of_year
5. Монотонный `time_idx` per-station (cumcount)
6. log1p преобразование всех 12 целевых переменных + shop_total_revenue
7. Z-score нормализация per-station (кроме статических, бинарных и целевых)
